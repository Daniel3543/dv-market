# DV MARKET - Premium Fruit & Vegetable Store

A modern e-commerce platform for fresh fruits and vegetables built with the MERN stack.

## Features

- 🛍️ Modern, responsive design with premium UI/UX
- 🔐 User authentication with JWT
- 🛒 Shopping cart functionality
- ❤️ Wishlist/Favorites system
- 💳 Stripe payment integration
- 📦 Order history and tracking
- 🔍 Product search and filtering
- 📱 Mobile-first responsive design
- ⚡ Lazy loading for images
- 🎯 SEO optimized

## Tech Stack

**Frontend:**
- React.js (Hooks)
- React Router DOM
- Tailwind CSS
- Framer Motion (Animations)
- React Query (Data fetching)
- Axios
- Stripe.js

**Backend:**
- Node.js
- Express.js
- MongoDB with Mongoose
- JWT Authentication
- Stripe API
